Loading pagespec.js

Lists a PageTask queue. The queue is checked against the page cookie / local store, amending where needed. Specifies a polling interval for fetching new tasks, and starts an interval that checks the pagespec server. Checking the server can be done by reloading the script, incrementing a search cachebuster parameter. Puts an id on the script tag 'pagespec_script'. Adds attribute 'polling' containing the polling interval timer id.

Checks the page for embedded tasks (only when page is loaded), adding them to the queue. For script tags that need translation, a dynamic script tag 'translated.js' is added encoding the source text to translate. If anything is in the queue a new script tag is added for loading pagetasks.js

Loading pagetasks.js

runs the outstanding tasks with the appropriate apis loaded in scope.

Loading translated.js

Takes the source script as a search parameter, returning the translated version in a scope or string that is either sent to an event handler or worker.
